Encryption and Decryption System



Description

A Java-based application that encrypts and decrypts text.

The system allows users to manage multiple keys, secure access with a PIN, and process sentences through a simple menu interface.



 Features

- PIN protection system

-  Create and manage multiple encryption keys

-  Encrypt text using selected keys

-  Decrypt text back to original form

-  Handle multiple sentences

-  Input validation for keys and PIN
